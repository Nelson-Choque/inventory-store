## Instalación

1. Clona este repositorio:

   ```shell
   git clone https://github.com/Nelson-Choque/inventory-store.git
   ```

2. Instala dependencias

   ```shell
   npm install
   ```

## Uso

1.- iniciar la aplicacion

```shell
npm start
```

2.- iniciar modo desarollo

```shell
npm run dev
```
